
# Dockord — le centre de contrôle Discord manquant

Une app Electron qui transforme Discord en quelque chose de lisible, organisé et efficace.

## Fonctionnalités
- **Sauvegarde manuelle** : Enregistrez vos liens Discord avec des notes.
- **Organisation** : Utilisez des tags et des collections pour trier vos informations.
- **Rappels** : Planifiez des notifications pour ne rien oublier.
- **Historique local** : Gardez vos traces même si Discord change.

## Développement
- `npm start` : Lancer l'application.
- `npm run dist` : Créer un exécutable.
